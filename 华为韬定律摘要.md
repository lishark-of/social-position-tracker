# 华为“韬定律”技术路线与可投资方向摘要

## 一句话结论

这份报告的核心判断是：华为“韬定律”不是简单追求更先进制程，而是把优化目标从“几何缩微”转向“时间缩微”，即通过缩短信号路径、降低访存延迟、减少跨节点通信开销，提升系统级性能。因此，真正更值得关注的投资机会，不是单押芯片本体，而是围绕互连、封装、光通信、EDA 和 AI 基础设施的外溢受益链。

## 核心逻辑

1. 技术目标变了  
过去拼的是晶体管更小、制程更先进；现在更重要的是让数据走得更短、更快、更省电。

2. 华为路线是系统级打法  
它不是只做单颗芯片，而是从器件、电路、芯片到系统一起优化，重点落在逻辑折叠、统一总线、统一内存编址和超节点光互联。

3. 手机是验证场，AI 服务器是主战场  
折叠手机和麒麟路线更像技术样板，真正的大规模资本开支仍在 AI 服务器、数据中心和高速互连。

## 技术路线拆解

### 1. Logic Folding

通过三维堆叠、缩短布线、提升带宽密度，缓解传统平面芯片的时延和功耗问题。  
这一部分更映射到混合键合、热设计、EDA、先进封装能力。

### 2. Unified Bus / 统一内存池

这是当前成熟度最高、也最接近产业兑现的一环。  
它的本质是把多节点、多芯片的内存资源做统一编址和高效调度，直接受益方向包括：

- 内存接口芯片
- CXL 类互连
- retimer / switch
- 服务器主板与载板
- 系统软件栈

### 3. 光互联 / Hi-ONE

面向高速集群通信，长期方向是硅光和共封装光学。  
但近两年更可能先兑现业绩的，仍然是 800G / 1.6T 可插拔光模块和其前置器件，而不是立刻全面落地 CPO。

## 时间线判断

- 2026 年：麒麟率先导入 Logic Folding
- 2026 年第四季度：Ascend 950DT 与 Atlas 950 超节点上市
- 2027 年第四季度：Atlas 960 超节点上市
- 2030 年前后：Logic Folding 更深入进入 AI 加速器
- 2035 年前后：真正意义上的 3D Folding 才可能成为主路径

## 投资映射

### 近两年优先关注

- 光模块与光器件
- 内存接口 / CXL 类互连
- 先进封装 OSAT
- 高速 PCB / 载板

### 两到四年重点关注

- 混合键合
- CPO
- EDA / IP
- 高阶热管理

### 更长期的小仓位方向

- 真正的 3D Folding
- 国产 HBM
- 前沿 AI 芯片弹性标的

## 组合思路

### 保守型

优先配置确定性更高、现金流更强的方向，围绕“光互联 + 封装 + 互连”布局。

### 平衡型

同时覆盖互连、封装、光模块和部分 EDA，兼顾兑现速度与技术弹性。

### 激进型

用较小仓位押注 CPO、混合键合、3D Folding、国产 HBM 等高赔率方向，但要接受更长兑现周期和更大波动。

## 最重要的落地结论

如果只压缩成一句话：  
先买“灵衢受益链”和“光互联现金流”，再用小仓位押注 3D Folding 与 CPO。
